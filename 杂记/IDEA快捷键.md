# IDEA快捷键

快捷键|作用
|:--:|:--:
Alt + Enter|快速导入包
Ctrl + O|快速覆写父类或接口中的方法
Alt + Insert|快速创建set和get方法
Alt + Insert|快速创建构造函数
Alt + F4|关闭IDEA
Alt + Insert|快捷键覆写方法
Alt + 1|选中单词，快速查看单词的意思，需要下载插件Translation
Ctrl + Alt + L|快速格式化代码
